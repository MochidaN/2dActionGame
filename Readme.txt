■ゲームの概要
2Dの横スクロールゲームで奥にいる黒色の敵を倒すとゲームクリア
プレイヤーのHPが0になるとゲームオーバー
━━━━━━━━━━━━━━━━━━━━
■操作方法
・キーボードの場合
[矢印キー左右]    ：移動
[S]    　　　　　 ：ステップ
[Z]               ：ジャンプ
[W]               ：ガード
[A]               ：攻撃

[矢印キー上下]    ：カーソル移動
[Enter]           ：決定

・ジョイステックの場合
[方向キー左右]    ：移動
[〇ボタン]        ：ステップ
[×ボタン]        ：ジャンプ
[L1ボタン]        ：ガード
[R1ボタン]        ：攻撃

[方向キー上下]    ：カーソル移動
[〇ボタン]        ：決定
